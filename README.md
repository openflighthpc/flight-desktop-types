# Flight Desktop Types

A collection of desktop session types for use with Flight Desktop.

## Overview

These types are for use with Flight Desktop and provide definitions of
a number of commonly used GUI desktop environments.

## Installation

This collection should be used with an existing functional
installation of Flight Desktop. Please refer to Flight Desktop
project for information on how to install.
